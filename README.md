# AI Essentials – Python

You don't need a framework to experiment with AI.

This notebook shows that you can interact with a large language model with a few lines of Python.
The model runs locally using ([Ollama](https://ollama.com)).

For a more detailed description, check out my [article](https://medium.com/@kinneko-de/e5e1e838ae75)

## Run the notebook

Open [`01_call_llm.ipynb`](01_call_llm.ipynb) directly in VS Code. The [Jupyter extension](https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter) is needed to run the code.
